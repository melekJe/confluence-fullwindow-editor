Fullwindow Editor

A toggle button for the editor, for a full window mode in confluence

Download .zip Download .tar.gz View on GitHub

Features

Adds a button to the toolbar of the editor, to toggle to a full window mode.

  • For commenting,
  • Pages and Blogposts
  • Supports quick commenting.
